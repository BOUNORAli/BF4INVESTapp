# Logo BF4 INVEST

Placez votre fichier image du logo dans ce répertoire.

## Formats supportés
- PNG (recommandé) : `logo.png`
- JPG/JPEG : `logo.jpg` ou `logo.jpeg`
- GIF : `logo.gif`

## Nom du fichier
Le fichier doit s'appeler **`logo.png`** (ou une autre extension supportée).

## Emplacement
Le logo sera automatiquement chargé depuis `backend/src/main/resources/images/logo.png` lors de la génération des PDF.

## Note
Si aucun fichier logo n'est trouvé, le système utilisera automatiquement un logo dessiné programmatiquement avec les mêmes caractéristiques visuelles (fond noir, lignes courbes bleues, texte "BF4 INVEST").
